## 🚀 Introdução

Python é uma linguagem de programação simples, poderosa e muito utilizada no mercado. Sua sintaxe clara facilita o aprendizado para iniciantes.

---

## 🏗️ Estrutura Básica

Um programa Python é executado linha por linha:

```python
print("Olá, mundo!")
```

## 📦 Variáveis

```python
nome = "Ana"
idade = 20
altura = 1.65
```

✔ Tipagem automática

## ⌨️ Entrada de Dados

## 🔄 Conversão de Tipos

```python

```

## ✅ Boas Práticas

- Use nomes claros
- Evite acentos
- Utilize `snake_case`

```python
media_final = 7.5
total_alunos = 30
```
